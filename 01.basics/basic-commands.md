# Linux Basic Commands

This file contains important basic Linux commands for beginners.

---

## 1. pwd

### Purpose
Shows the current working directory.

### Syntax
pwd

### Example
pwd

### Output
/home/rutuja

---

## 2. ls

### Purpose
Lists files and directories.

### Syntax
ls

### Examples
ls
ls -l
ls -a

### Explanation
- ls → show files
- ls -l → detailed list
- ls -a → hidden files

---

## 3. cd

### Purpose
Change directory.

### Syntax
cd directory_name

### Example
cd Documents

### Special Commands
cd ..   → go back one directory  
cd ~    → go to home directory  

---

## 4. mkdir

### Purpose
Create a new directory.

### Syntax
mkdir folder_name

### Example
mkdir projects

---

## 5. rmdir

### Purpose
Remove empty directory.

### Syntax
rmdir folder_name

### Example
rmdir test

---

## 6. touch

### Purpose
Create empty file.

### Syntax
touch file_name

### Example
touch notes.txt

---

## 7. cp

### Purpose
Copy files or directories.

### Syntax
cp source destination

### Example
cp file1.txt file2.txt

---

## 8. mv

### Purpose
Move or rename files.

### Syntax
mv source destination

### Example
mv file1.txt Documents/

---

## 9. rm

### Purpose
Delete files.

### Syntax
rm file_name

### Example
rm test.txt

### Warning
This command permanently deletes files.

---

## 10. clear

### Purpose
Clear terminal screen.

### Syntax
clear

---

## 11. whoami

### Purpose
Shows current logged-in user.

### Syntax
whoami

---

## 12. uname

### Purpose
Shows system information.

### Syntax
uname -a

---

# Summary

These commands are the foundation of Linux.

- pwd
