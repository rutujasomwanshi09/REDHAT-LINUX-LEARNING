# Linux System Information Commands

This document explains commands used to check system information in Linux.

---

## 1. uname

Displays system information.

Example:
uname -a

---

## 2. hostname

Displays the system hostname.

Example:
hostname

---

## 3. uptime

Shows how long the system has been running.

Example:
uptime

---

## 4. free

Displays memory usage.

Example:
free -h

---

## 5. lscpu

Shows CPU architecture information.

Example:
lscpu

---

## 6. lsblk

Lists block devices and storage information.

Example:
lsblk

---

## Summary

Important Linux system information commands:

- uname
- hostname
- uptime
- free
- lscpu
- lsblk
