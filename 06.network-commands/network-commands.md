# Linux Network Commands

This document explains commonly used Linux networking commands for checking network connectivity and configuration.

---

## 1. ping

Checks network connectivity between systems.

Example:
ping google.com

---

## 2. ifconfig

Displays network interface configuration.

Example:
ifconfig

---

## 3. ip

Shows and manages network interfaces.

Example:
ip addr

---

## 4. netstat

Displays network connections and statistics.

Example:
netstat -tuln

---

## 5. traceroute

Shows the path packets take to reach a destination.

Example:
traceroute google.com

---

## 6. nslookup

Used to query DNS information.

Example:
nslookup google.com

---

## 7. hostname

Displays or sets the system hostname.

Example:
hostname

---

# Summary

Important Linux networking commands:

- ping
- ifconfig
- ip
- netstat
- traceroute
- nslookup
- hostname
