# Linux Process Management Commands

This document explains commands used to monitor and manage processes in Linux.

---

## 1. ps

Displays information about running processes.

Example:
ps

Detailed view:
ps -ef

---

## 2. top

Shows real-time running processes and system resource usage.

Example:
top

---

## 3. htop

Interactive process viewer (improved version of top).

Example:
htop

---

## 4. kill

Terminates a process using its process ID (PID).

Syntax:
kill PID

Example:
kill 1234

---

## 5. pkill

Kills processes using the process name.

Example:
pkill firefox

---

## 6. bg

Runs a process in the background.

Example:
bg

---

## 7. fg

Brings a background process to the foreground.

Example:
fg

---

# Summary

Important Linux process management commands:

- ps
- top
- htop
- kill
- pkill
- bg
- fg
