# Linux System Monitoring Commands

This document explains commands used to monitor system performance in Linux.

---

## 1. top

Displays real-time system processes and CPU usage.

Example:
top

Press q to exit.

---

## 2. htop

Interactive process viewer (better version of top).

Example:
htop

Note: htop may need to be installed first.

---

## 3. vmstat

Displays memory, CPU, and process statistics.

Example:
vmstat

---

## 4. iostat

Displays CPU and disk input/output statistics.

Example:
iostat

---

## 5. netstat

Displays network connections and statistics.

Example:
netstat -tuln

Shows listening ports.

---

# Summary

Important Linux system monitoring commands:

- top
- htop
- vmstat
- iostat
- netstat
