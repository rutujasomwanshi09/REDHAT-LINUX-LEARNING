# Log Analyzer Tool

This project is a Bash script that analyzes log files and generates a summary report.

## Features

- Counts total lines
- Counts ERROR, WARNING, INFO logs
- Shows top errors
- Displays last 5 log entries

## How to Run

chmod +x log-analyzer.sh
./log-analyzer.sh

## Technologies Used

- Linux
- Bash scripting
- grep, wc, sort
